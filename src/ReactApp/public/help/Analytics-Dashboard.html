Analytics Dashboard

Analytics Dashboard

The Analytics tab allows users to perform analytics on past data that are useful for decision making. Please, hover over the different plots in the Analytics tab to get a detailed description of each plot.

On the right part of the screen, users can find a menu with all the displayed plots. Clicking any of the options of the menu will take you to the corresponding plot.

Note that the Analytics tab is connected to the Documents tab. Data shown in the dashboards corresponds to the list displayed in the documentation schedule. If we apply a constraint through a set of rules in the search bar, the same rules will be shown in the Analytics tab, and plotted data will only correspond to this subset of documents. Analogously, if we apply a constraint in the search bar in the Analytics tab, when we click the Documents tab we will only see the list of documents corresponding to the subset of documents resulting from applying this constraint. In other words, the search tool remains consistent across tabs.

In order to know more about the search capabilities of our system, please refer to the Documents tab section's help option.

We would like to note that this feature is powerful for analytics as you can focus your analysis to any interesting subset of documents at any time.
